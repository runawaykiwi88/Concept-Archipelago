Before We Set Sail

This game uses the Anthropic API to judge the quality of your connections. Enter your API key to begin — it stays in your browser and is never stored.


How to get your API key

  1. Visit console.anthropic.com
  2. Sign up or log in to your Anthropic account
  3. Go to API Keys in the left sidebar
  4. Click Create Key, give it a name
  5. Copy the key (it starts with sk-ant-) and paste it above

Note: API calls cost a small amount of credit. Each bridge crossing uses roughly $0.001.
